## 📌 Project Overview

The **Contact Book Application** is a console-based Python project developed as part of my **Python Programming Internship at CodSoft**.

This application allows users to manage personal or professional contacts efficiently through a structured, menu-driven interface. It demonstrates fundamental Python programming concepts such as functions, loops, conditional statements, lists, dictionaries, and user input handling.

The application supports adding, viewing, searching, updating, and deleting contact records during runtime.

---

## 🚀 Features

### ➕ Add New Contacts
- Name
- Phone Number
- Email Address
- Physical Address

### 📋 View Contacts
- Display all saved contacts

### 🔍 Search Contacts
- Search by name
- Search by phone number

### ✏ Update Contacts
- Modify phone number, email, or address

### 🗑 Delete Contacts

## 🎯 How the Application Works

The user selects an option from the main menu:

1. Add Contact
2. View Contacts
3. Search Contact
4. Update Contact
5. Delete Contact
6. Exit

Based on the selected option, the program performs the requested operation. All contacts are stored in memory using a list of dictionaries during runtime. The application continues running until the user selects **Exit**.
